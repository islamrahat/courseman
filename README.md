# CourseMan  
### An API for Managing FSE Courses

CourseMan is a RESTful API developed using Java Spring Boot to manage academic course information for the Faculty of Science and Engineering (FSE).

## Project Overview

CourseMan provides a lightweight and structured way to manage course-related data for FSE.  
The system focuses on clarity, clean API design, and in-memory data handling, making it suitable for academic use and demonstration purposes.

---

## Technology Stack

- Java 25  
- Spring Boot 4.1  
- Maven  
- RESTful API architecture  
- In-memory data storage (no external database)

---

## Course Entity Details

Each course managed by the API includes the following attributes:

| Field | Description |
|------|------------|
| `id` | Auto-incremented, starting from **100** |
| `code` | Course code |
| `title` | Course title |
| `credit` | Course credit |
| `type` | Theory / Lab |
| `semester` | Autumn / Spring (202x) |
| `departmentId` | 10 |
| `teacher` | Course instructor |

---

## Features

- RESTful API design
- Create, read, update, and delete (CRUD) course information
- Auto-generated course IDs
- Clean and modular Spring Boot structure
- In-memory data handling (no database dependency)
